# OMNIEVENTS BACKEND APPLICATION:

## This is the omnievents node.js/express.js backend application.

* NodeJS version:  v20.20.1

## Install & Run steps:

* npm install
* npm start

## DB Select:

* Assign 'local' to DB_PROVIDER in .env file, for local PostgrSQL DB use.
* Assign 'neon' to DB_PROVIDER in .env file, for NEON DB use.
